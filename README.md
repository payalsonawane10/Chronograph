# ChronoGraph

## AI-Powered Temporal GraphRAG for Enterprise Knowledge

### Project Overview
ChronoGraph is an AI-powered application that helps users explore enterprise knowledge using a temporal knowledge graph. It connects information from multiple sources, understands relationships over time, and provides accurate answers with timeline-based context.

### Features
- AI-powered Question Answering
- Temporal Knowledge Graph
- Timeline Visualization
- Enterprise Data Analysis
- GraphRAG Integration
- Interactive Chat Interface

### Technology Stack
- React
- Next.js
- Python
- FastAPI
- Neo4j
- LangChain
- LlamaIndex
- Ollama / GPT
